# 石锅王图片生成提示词模板

## 淘宝主图提示词

### 基础版（文生图）
```
云南天然石材手工制作的蒸汽石锅，锅底正中央只有一个圆形蒸汽孔（只有一个孔！），
深灰色天然石材纹理，白色纯净背景，产品居中，专业电商摄影打光，高清，800x800方图。
```

### Seedream 专用版（推荐）
```
Yunnan handmade natural stone steam pot, with exactly ONE single round steam hole at the bottom center,
dark grey natural stone texture, white background, centered, professional product photography, 
sharp focus, high quality, 1920x1920
```

### 带文案版
```
云南蒸汽石锅电商主图，天然石材，锅底有蒸汽孔。
白色背景，产品居中，右上角添加文字"天然石材"，左下角添加"原汁原味"。
专业电商摄影风格，清晰，高质量。
```

### 图生图版（提供参考图时使用）
```
以参考图片的石锅为基础，保持石锅的真实形状和材质不变（特别是锅底的蒸汽孔）。
重新设计背景为专业电商风格，添加产品光影效果。
不要改变石锅本身的外观，只优化整体构图和光线。
输出电商主图风格，800x800。
```

## 调用 Nano Banana 的格式

```python
# 文生图
payload = {
    "model": "flux-pro",  # 或 stable-diffusion-xl
    "prompt": "上面的提示词",
    "width": 800,
    "height": 800,
    "steps": 30,
    "guidance_scale": 7.5
}

# 图生图（需要提供参考图）
payload = {
    "model": "flux-pro",
    "prompt": "图生图提示词",
    "init_image": "base64编码的图片或URL",
    "strength": 0.7,  # 0=完全保留原图，1=完全重新生成
    "width": 800,
    "height": 800
}
```
